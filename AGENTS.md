# AGENTS.md

## Project Overview

- This project is an Astro-based personal homepage.
- The final deployment target is a GitHub Pages `github.io` address.

## Working Rules

- Do not add new libraries without the user's approval.
- Do not modify unrelated files.
- Do not write secrets, tokens, passwords, or API keys in code.
- Always ask before running delete commands.
- After making changes, explain which files were changed and why.
- When possible, verify changes with `npm run build`.
- Commit changes frequently when there are project modifications.

## Commit Message Rules

Use one of the following commit message formats:

- `feat: 한국어설명`
- `fix: 한국어설명`
- `chore: 한국어설명`
- `docs: 한국어설명`
